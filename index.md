# Whisker Maven repo

This branch hosts the `rs.whisker` Maven 2 layout served at
<https://whiskerrs.github.io/whisker/maven/>.

Currently the only artifact published here is the
`rs.whisker:whisker-gradle-plugin` (and its Gradle plugin marker
`rs.whisker.gradle:rs.whisker.gradle.gradle.plugin`).

See `.github/workflows/publish-gradle-plugin.yml` on `main` for the
publishing pipeline.
